# eXeMeLeR

Generate **Resolume Arena Advanced Output** screen-setup XML (polygon slices + masks) from
decorative patterns, imported SVG/PDF/EPS artwork, and Boolean-op results — a native macOS app.

## Download

Grab the latest signed & notarized **`eXeMeLeR-1.0.0.dmg`** from
[**Releases**](https://github.com/AmritusG/exemeler-release/releases/latest).

Open the DMG and drag **eXeMeLeR** to **Applications**. The build is signed with a Developer ID
and notarized by Apple, so it launches with no Gatekeeper warning.

## Requirements

- macOS 14.0 (Sonoma) or later
- Universal binary (Apple Silicon + Intel)

## Using the output

Generated `.xml` screen setups load in Resolume via **Output ▸ Advanced… ▸ Presets ▸ Load…**
(place files in `~/Documents/Resolume Arena/Presets/Advanced Output/`).
